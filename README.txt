# Instructions on executing code

parser.py
=========
	
Executing 'python parser.py' will scrape xml data from Wikipedia and parse into SQL database


searcher.py
===========

Executing 'python searcher.py' will use pre-existing index if one exists else it will create one and then allow you to search it.

Executing 'python searcher.py redo' will re-index the database (overwriting old index) and then allow you to search it.
